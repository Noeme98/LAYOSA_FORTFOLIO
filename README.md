# Neil Francis G. Loyosa - Portfolio Website

A comprehensive, modern portfolio website built with HTML, CSS, and JavaScript.

## Features

- **Hero Section**: Eye-catching introduction with call-to-action
- **About Section**: Personal introduction with profile placeholder
- **Skills Section**: Interactive skill bars with proficiency levels
- **Projects Section**: Showcase of key projects with tech stacks and links
- **Experience Section**: Professional timeline with work history
- **Contact Section**: Contact form and information with social links
- Responsive design that works on desktop and mobile
- Smooth scrolling navigation
- Scroll-triggered animations
- Typewriter effect on hero subtitle
- Hover effects and modern UI elements

## Sections

1. **Hero**: Welcome message and introduction
2. **About**: Personal background and photo placeholder
3. **Skills**: Technical skills organized by category with progress bars
4. **Projects**: Featured projects with descriptions, tech stacks, and links
5. **Experience**: Work history in timeline format
6. **Contact**: Contact form, information, and social media links

## How to View

1. Open `index.html` in your web browser.
2. Alternatively, serve the files using a local web server for better functionality.

## Structure

- `index.html` - Main HTML file with all sections
- `css/styles.css` - Comprehensive styling with modern design
- `js/script.js` - JavaScript for interactivity and animations

## Customization

- **Personal Info**: Update name, bio, contact details in `index.html`
- **Skills**: Modify skill levels and categories in the Skills section
- **Projects**: Replace project descriptions, tech stacks, and add real links
- **Experience**: Update work history and dates
- **Colors**: Adjust color scheme in `css/styles.css`
- **Images**: Replace placeholders with actual images (profile, project screenshots)

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6+)
- Responsive Design
- Modern UI/UX Practices
